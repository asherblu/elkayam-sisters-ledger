# חשבונות בין האחיות - משפחת אלקיים

אפליקציית PWA למעקב חיובים והחזרים כספיים בין ריקי, ברוריה ודורית, עם יתרה שוטפת לכל זוג, יומן פעולות, פתקיות אישיות ומשותפות, והתראות בכניסה.

## הרצה מקומית

פתחו את `index.html` ישירות בדפדפן, או הריצו שרת סטטי מקומי (למשל `python -m http.server`) כדי שה-Service Worker יפעל כראוי.

## מצב נוכחי

מחוברת ל-Firebase (Firestore + Authentication) - הנתונים מסונכרנים בזמן אמת בין כל המכשירים של שלוש האחיות. כניסה עם שם + סיסמה (מאחורי הקלעים ממופה לאימייל סינתטי קבוע לכל אחות וכניסה אמיתית דרך Firebase Auth). חוקי האבטחה ב-`firestore.rules` מגבילות גישה לשלוש החשבונות הידועות בלבד.
