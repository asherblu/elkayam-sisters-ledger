# חשבונות בין האחיות - משפחת אלקיים

אפליקציית PWA למעקב חיובים והחזרים כספיים בין ריקי, ברוריה ודורית, עם יתרה שוטפת לכל זוג, יומן פעולות, פתקיות אישיות ומשותפות, והתראות בכניסה.

## הרצה מקומית

פתחו את `index.html` ישירות בדפדפן, או הריצו שרת סטטי מקומי (למשל `python -m http.server`) כדי שה-Service Worker יפעל כראוי.

## מצב נוכחי

גרסת פרוטוטייפ - הנתונים נשמרים ב-localStorage בדפדפן בלבד (ללא סנכרון בין מכשירים). חיבור ל-Firebase (Firestore + Auth) מתוכנן כשלב הבא.
